# 🔮 Crystal Music Player

实时音频驱动的三维交互体验。上传任意音频，低频触发碎片爆炸，响度控制泛光，高频偏移色相——将声音的每一个瞬间实体化为视觉动画。

## Tech Stack

- **Next.js** + **React Three Fiber** + **Three.js WebGPU**
- **Rapier** 物理引擎
- **Web Audio API** 实时频谱分析

## Getting Started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # 生产构建
```

## Features

- 鼠标拖拽控制水晶球自转方向与惯性
- 低频 Bass 驱动碎片爆炸冲量
- 整体响度映射 Bloom 泛光强度
- 高频 Treble 偏移材质色相
- 支持 MP3 · WAV · FLAC · AAC · M4A · MP4 上传
